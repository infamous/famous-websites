Famo.us 0.3 Reference Documentation

Welcome to the reference documentation for Famo.us 0.3. Here, you’ll find everything you need to know about Famo.us, from the core platform to support tools to third-party integrations.

Preface

This reference documentation is intended for programmers who want to build applications with the Famo.us platform. It includes explanations and live-coding examples that cover how to use the essential bits and pieces.

If you’re a beginner who is looking for resources on learning to program, see the resources page. If you’re an expert programmer interested in low-level details about the platform, see the API documentation or the source code.

Getting help

The best place to get help is our community chat room. If you have any problems at all with this documentation, don’t hesitate to join the chat room and ask. The Famo.us staff monitors it regularly and will be there to answer your questions, no matter how simple or complex.

Found a problem? Caught a typo?

If you find anything wrong with this reference documentation, let us know at bugs@famo.us. If you have a suggestion for improving it, email us at suggestions@famo.us.
